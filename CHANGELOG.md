# История изменений (Changelog)

Все значимые изменения в этом проекте документируются в данном файле. Формат файла основан на принципах [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/), и проект следует семантическому версионированию [SemVer](https://semver.org/spec/v2.0.0.html).

## [1.0.0] — 2026-06-07

Это первый стабильный релиз системы мониторинга и анализа цен конкурентов.

### Добавлено
- **Парсер Космеи (`scraper.py`)**: Сбор информации об услугах и ценах с сайта конкурента с помощью BeautifulSoup4.
- **Парсер Формы (`scraper_forma.py`)**: Сбор информации об услугах и ценах собственного салона на платформе Bazium.
- **Интеграция с Telegram (`telegram_utils.py`)**: Автоматическая отправка уведомлений при изменении цен конкурента (Космея) и нашего собственного салона (Форма).
- **FastAPI Бэкенд (`main.py`)**:
  - Endpoint `/api/parser/run` для фонового запуска скрапинга.
  - Endpoint `/api/parser/results` и `/api/parser/forma-results` для получения данных.
  - Endpoint `/api/parser/comparison` для сопоставления услуг с использованием алгоритма `difflib`.
  - Endpoint `/api/parser/history` для отслеживания истории цен.
- **Интерактивный дашборд React (Vite)**:
  - Панели со статистикой (общее число услуг, сопоставленные позиции, резервы для роста цен).
  - Списки услуг с фильтрацией по категориям и текстовым поиском.
  - Удобная таблица сравнения с подсветкой ценовой разницы и рекомендациями.
  - Графики цен на базе `Recharts` для отображения динамики изменения цен в модальном окне услуги.
  - Функция экспорта текущей вкладки в файл CSV.
- **Конфигурация репозитория**: Добавлены файлы документации `README.md`, `CHANGELOG.md` и файл `.gitignore`.
