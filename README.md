# PriceMonitor — Система мониторинга цен конкурентов

**PriceMonitor** — это полноценное аналитическое решение для автоматического парсинга, сопоставления и анализа цен услуг салонов красоты в г. Волгодонск. Система сравнивает цены салона-конкурента **«Космея»** с ценами нашего салона **«Форма»**, выявляет расхождения и помогает находить резервы для оптимизации ценообразования.

## 📋 Основные функции

- **Мультиресурсный парсинг**: Сбор данных с сайтов салонов красоты «Космея» и «Форма» в режиме реального времени.
- **Интеллектуальное сопоставление (Fuzzy Matching)**: Нечеткое сравнение названий услуг с использованием алгоритмов сходства строк (`difflib.SequenceMatcher`) для автоматического нахождения пар одинаковых или похожих услуг.
- **Уведомления в Telegram**: Мгновенная отправка красивых HTML-сообщений в Telegram-канал/чат при обнаружении изменений в ценах конкурента.
- **Интерактивный дашборд**:
  - Обзор услуг конкурента и собственного салона с фильтрацией по категориям и текстовым поиском.
  - Таблица сравнения цен с подсветкой разницы в стоимости и рекомендациями (например, «Резерв для повышения»).
  - Анализ упущенной выгоды и расчет количества услуг, цены на которые можно поднять.
  - Графики истории изменения цен по каждой услуге с использованием библиотеки `Recharts`.
  - Возможность быстрого экспорта отчетов в формат CSV.

## 🛠️ Технологический стек

### Бэкенд (Backend)
- **Язык**: Python 3.10+
- **Фреймворк**: FastAPI + Uvicorn (асинхронный веб-сервер)
- **Парсинг**: Beautiful Soup 4 + Requests
- **Уведомления**: Telegram Bot API
- **База данных**: Локальные JSON-файлы с исторической структурой
- **Сопоставление**: Модуль `difflib` (встроенный в Python)

### Фронтенд (Frontend)
- **Библиотека**: React 18+ (Vite)
- **Иконки**: Lucide React
- **Интерактивные графики**: Recharts
- **Стилизация**: CSS3 с кастомными переменными (темная премиум-тема)

## 🚀 Быстрый запуск

### Шаг 1: Клонирование и настройка окружения

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/vsanyanov-ux/price-scraper.git
   cd price-scraper
   ```

2. Настройте конфигурацию бэкенда. Создайте файл `backend/.env` на основе примера:
   ```bash
   cp backend/.env.example backend/.env
   ```
   Заполните параметры в `.env`:
   ```env
   TELEGRAM_BOT_TOKEN=your_bot_token_here
   TELEGRAM_CHAT_ID=your_chat_id_here
   ```

---

### Шаг 2: Запуск бэкенда

1. Перейдите в директорию `backend`:
   ```bash
   cd backend
   ```
2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv .venv
   # Для Windows:
   .venv\Scripts\activate
   # Для macOS/Linux:
   source .venv/bin/activate
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите сервер разработки FastAPI:
   ```bash
   python main.py
   ```
   API будет доступно по адресу: `http://127.0.0.1:8000`  
   Документация Swagger (Swagger UI): `http://127.0.0.1:8000/docs`

---

### Шаг 3: Запуск фронтенда

1. Откройте новое окно терминала в корне проекта и перейдите в `frontend`:
   ```bash
   cd frontend
   ```
2. Установите npm-зависимости:
   ```bash
   npm install
   ```
3. Запустите фронтенд-сервер разработки:
   ```bash
   npm run dev
   ```
4. Откройте приложение в браузере по адресу: `http://localhost:5173`

## 📈 Схема данных

Данные сохраняются в файлы `scraped_data.json` и `forma_data.json`.
При изменении цены старое значение записывается в массив `history` с указанием даты изменения, что позволяет строить график колебаний стоимости в реальном времени.

Пример структуры записи:
```json
"Парикмахерские услуги|||Стрижка женская": {
    "title": "Стрижка женская",
    "price_raw": "1200 руб",
    "price_value": "1200",
    "category": "Парикмахерские услуги",
    "url": "https://...",
    "history": [
        {
            "price_raw": "1000 руб",
            "date": "2026-05-15T12:00:00"
        }
    ],
    "last_updated": "2026-06-07T18:00:00",
    "price_changed": true
}
```
